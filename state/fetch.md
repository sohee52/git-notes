## `git fetch`

### 역할

* **원격 저장소의 최신 커밋 정보를 로컬로 가져온다.**
* 로컬 브랜치에는 **아무 변화도 주지 않는다.**

### 상태 변화

* `origin/main`, `origin/dev` 같은
  **원격 추적 브랜치만 업데이트됨**
* 현재 작업 중인 브랜치의 코드, HEAD, 스테이징 영역은 **변경 없음**

### 영향 받는 영역

* ❌ 작업 디렉토리
* ❌ 스테이징 영역
* ❌ 현재 브랜치
* ✅ 원격 추적 브랜치 (`origin/*`)

### 사용 상황

* 원격에 어떤 변경이 있는지 **미리 확인**하고 싶을 때
* 머지나 pull 전에 **안전하게 차이만 확인**하고 싶을 때

### 관련 명령

```bash
git fetch
git log HEAD..origin/main
git diff HEAD..origin/main
```

### 요약

> **원격 변경 사항만 가져오고, 로컬은 그대로 둔다**
