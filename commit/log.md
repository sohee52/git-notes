**그동안 커밋한 내용 알아보는 방법**

| **목적** | **명령어** | **비고** |
| --- | --- | --- |
| **전체 흐름 파악 (강력 추천)** | `git log --oneline --graph --decorate` | 브랜치 관계를 그래프로 보여줌 |
| **내가 한 커밋만 보기** | `git log --author="내이름"` | 이름이나 이메일로 필터링 |
| **최근 커밋 내용(코드) 보기** | `git log -p -<숫자>` | 예: `git log -p -3` (최근 3개) |
| 특정 파일의 이력 보기 | `git log -- <파일이름>` | 해당 파일만 필터링 |
